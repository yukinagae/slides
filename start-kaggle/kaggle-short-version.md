# ==2.== kaggleの仕組み

---

## 大まかな流れ

1. 主催者（企業など）がコンペを主催する
	a. データを準備
	b. 問題を定義する
2. 参加者は様々な手法を使ってベストなモデルを構築し、予測を提出する
	* スコアやランキングが分かる
4. 主催者は、精度が高い予測に賞金を払う
